## What's the Problem?

Students in 2026 don't fail because information is missing — they fail because there's **too much of it**. Google, YouTube, PDFs, lecture slides — all explaining the same concept differently. The result? Cognitive overload, procrastination, and zero deep learning.

Coaching centers charge thousands and rush through syllabi. Slow learners get left behind. The average student loses focus after 8 minutes of a lecture. And when they try to self-study? They open YouTube "to learn" and end up watching shorts for 3 hours.

There's no accessible tool that can **structure an entire subject**, **explain concepts on demand**, **visualize how topics connect**, and **keep students engaged** — all from one place.

## How Vidya Solves It

**Vidya** (from Sanskrit "विद्या" — knowledge) turns any textbook into a complete, interactive learning experience:

1. **Maps your subject** — Enter any book. Nova extracts every chapter and topic automatically. A clear roadmap, no guessing.
2. **Teaches you** — 7 learning modes for every topic, from deep explanations to 60-second reels
3. **Answers your doubts** — Ask anything, get instant Nova-powered answers. No waiting for office hours.
4. **Shows connections** — A live knowledge graph reveals how concepts link together — the way experts think
5. **Scans your pages** — Photograph any textbook page or handwritten notes. Nova Vision reads and explains it instantly.
6. **Tests you** — Battle mode, MCQs, exam-focused content — learning through competition
7. **Keeps you focused** — Gamified Pomodoro with anti-sleep detection. No more "I'll do it tomorrow."

## What Makes It Special

- **7 learning modes per topic** — Deep Concept, Quick Revision, Exam Mode, Knowledge Graph, Questions, Reels, Battle
- **Scan Page (Nova Vision)** — Point your camera at any printed page, handwritten notes, or diagram. Nova Vision reads it and gives instant analysis. Physical paper becomes interactive learning.
- **Live Knowledge Graph** — D3.js force-directed graph showing how concepts interconnect, powered by Nova Embed similarity
- **Micro Reels** — 60-second TikTok-style learning cards generated by Nova
- **Battle Mode** — Compete against AI with real-time scoring
- **Focus Mode** — Gamified Pomodoro timer with anti-sleep detection
- **Doubt Engine** — Instant streaming answers to any question
- **Works with any textbook** — Not locked to specific content. Enter any subject and book.

## How It Works (The Flow)

1. You enter a subject and textbook — Nova Lite maps every chapter and topic
2. You click a topic — 7 learning modes appear, each powered by Nova
3. Deep Concept gives you first-principles content with curiosity hooks
4. Confused? Ask the Doubt Engine — Nova answers instantly with streaming
5. Click Knowledge Graph — Nova Embed calculates topic similarities, D3.js renders the visual network
6. Need a break? Reels serve 60-second micro-lessons to keep momentum alive
7. Ready to test? Battle Mode generates questions and scores you against AI
8. Have a printed page? Scan Page uses Nova Vision to read and analyze it instantly
9. Time to focus? Focus Mode runs a Pomodoro timer with anti-sleep detection
10. Gamification (XP, streaks, levels) keeps you coming back

## Quick Start

### What you need
- Python 3.11+
- An Amazon Nova API key (AWS Bedrock access)
- Any modern browser

### Setup

```bash
# Clone the repo
git clone https://github.com/kunalsdas/vidya-ai.git
cd vidya-ai

# Set up environment
cp .env.example .env
# Edit .env and add your NOVA_API_KEY

# Install dependencies
cd backend
pip install -r requirements.txt

# Run the server
uvicorn main:app --reload
```

Open **http://localhost:8000** in your browser. Enter any subject and book to start learning.

## Deploy to Render.com

### One-click deploy

The repo includes a `render.yaml` blueprint. Connect your GitHub repo on Render and it auto-configures everything. Just add your `NOVA_API_KEY` as an environment variable.

### Manual deploy

1. Create a new **Web Service** on Render
2. Connect your GitHub repo
3. Set **Root Directory** to `backend`
4. Set **Build Command** to `pip install -r requirements.txt`
5. Set **Start Command** to `uvicorn main:app --host 0.0.0.0 --port $PORT`
6. Add environment variable: `NOVA_API_KEY=your_key`
7. Deploy

## Why Vidya Stands Out

**Not a chatbot.** You never type prompts into a text box. Vidya structures your entire learning path and gives you 7 ways to master every topic — automatically.

**Three Nova models working together.** Lite generates all content. Embed powers semantic connections in the knowledge graph. Vision reads physical documents. They complement each other to create something none could do alone.

**Built for how students actually learn.** Short bursts (Reels), competition (Battle), visual connections (Knowledge Graph), instant answers (Doubt Engine). Every feature maps to real student behavior, not how we wish they'd study.

**Works with any textbook.** Not locked to specific content or curricula. Enter any subject, any book — Vidya maps it and teaches it.
